Detailed Description

Main interface for the WaveShaper Infernce Engine.

The WaveShaper Inference Engine provides developers with simple access to powerful audio processing AI models (such as MicUpgrade, SpectralEnhancement, etc). The WaveShaper Inference Engine is a lightweight and cross-platform library that can be easily integrated into applications. The library can load pre-trained AI models and use these for audio processing to perform various tasks.
